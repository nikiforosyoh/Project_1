# Project 1

This repository contains `nikiforos_portfolio.html` and a small test to ensure the file is valid HTML.

## Running the HTML validation test

1. Install the test dependencies (from a shell):
   ```bash
   pip install pytest html5validator
   ```
2. Run the test using `pytest`:
   ```bash
   pytest
   ```

The test uses `html5validator` to check that `nikiforos_portfolio.html` has no HTML validation errors.
